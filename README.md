## Tracking Robot - How To :

1. Télécharger PixyMon & les librairies Pixy pour Arduino : https://pixycam.com/downloads-pixy1/
2. Faire le branchement des deux arduinos vu sur master_slave.png
3. Upload master_arduino.ino sur la carte "maître" (relié à la caméra Pixy)
4. Upload slave_arduino.ino sur la carte "esclave" (relié au shield Moteur)
5. Brancher le Shield Pixy au PC par Mini-USB
6. Ouvrer PixyMon puis Action > Set Signature 1 et prendre l'objet en question
7. Allumer le moteur puis débrancher la caméra
8. Posez et amusez-vous
